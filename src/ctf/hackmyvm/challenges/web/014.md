# 014

## Description

Find the flag in the domain [http://momo.hackmyvm.eu](http://momo.hackmyvm.eu)

## Solution

My initial thoughts were to check DNS TXT records, but it were empty ([https://toolbox.googleapps.com/apps/dig](https://toolbox.googleapps.com/apps/dig)).

Website itself doesn't contain anything, so next I checked `robots.txt`
```bash
➜ curl http://momo.hackmyvm.eu -i
```
