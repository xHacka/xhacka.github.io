## Description

Obtain the flag here http://momo.hackmyvm.eu/0r1g1n/ 

> Hint: `hackmyvm.hmv`

## Solution

The path says `Origin`, combined with hint the header should be added.

Also I can't believe I winged the auth right away with `admin:admin` lmao, thought it would take more tries.
```bash
➜ curl http://momo.hackmyvm.eu/0r1g1n/

➜ curl http://momo.hackmyvm.eu/0r1g1n/ -H 'Origin: hackmyvm.hmv'

            <form method="post" action="">
                <label for="username">Username:</label>
                <input type="text" id="username" name="username" required>    
                <br>
                <label for="password">Password:</label>
                <input type="password" id="password" name="password" required>
                <br>
                <input type="submit" value="Login">
            </form>
```
