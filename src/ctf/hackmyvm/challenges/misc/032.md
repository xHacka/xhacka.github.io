If you have `ffmpeg` you can do:
```
ffmpeg -i input.gif frame_%03d.png
```

I didn't want to boot VM, so I used Pillow.
```python
from PIL import Image
from pathlib import Path

save_path = Path("frames")
save_path.mkdir(exist_ok=True)

im = Image.open("032.gif")
frame = 0
while True:
    im.save(save_path / f"frame_{frame:03d}.png")
    frame += 1
    try:
        im.seek(frame)
    except EOFError:
        break
```

You get 6 frames, assemble the flag and submit.
