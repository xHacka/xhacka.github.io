## Solution

Since Im powershell I'll need to replace `\` with Backtick so characters are escaped properly. The output will show color coded text.
```powershell
➜ Write-Host ('TEXT' -replace '\\033', [char]27)

➜ Write-Host ('HMV{\033[31mw\033[31mh\033[31my\033[31m i\033[31mm \033[31mn\033[31mo\033[31mt \033[39m m\033[39ma\033[39my\033[39mb\033[39me \033[39mu\033[39ms\033[39me \033[32m g\033[32mr\033[32me\033[32me\033[32mn\033[32mt\033[32mh\033[32me\033[32mf\033[32ml\033[32ma\033[32mg\033[39m}' -replace '\\033', [char]27)
```
