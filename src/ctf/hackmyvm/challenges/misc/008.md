# 008

## Description

Flag inside the zip file.  [Download](https://downloads.hackmyvm.eu/challenges/008.zip)

## Solution

```powershell
➜ curl -LOs https://downloads.hackmyvm.eu/challenges/008.zip
➜ 7z x .\008.zip
```
